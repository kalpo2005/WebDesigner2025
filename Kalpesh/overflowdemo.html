CSS Overflow Property Examples

Overflow: visible

This text will overflow and be visible.

Overflow: hidden

This text will be clipped.

Overflow: scroll

Scrollbars will always be shown, even if the content doesn't overflow.

Overflow: auto

Scrollbars will be shown only if the content overflows.

Overflow: overlay

Scrollbars will be shown only if the content overflows, and they don't take up space.

Overflow-x: hidden; Overflow-y: scroll;

This text will be clipped horizontally and scrollable vertically.

Example with Long Text (demonstrates wrapping and overflow)

This is a long paragraph of text designed to demonstrate how the overflow property works when the text is long enough to wrap and potentially overflow the container's boundaries. It will wrap within the container, but if it becomes too long vertically, a scrollbar will appear.
